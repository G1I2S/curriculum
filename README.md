# CV - Edglimar Espinoza

Curriculum Vitae personal desarrollado con HTML y Bootstrap 5.

## Descripción

Sitio web de CV profesional que incluye:
- Información de contacto
- Formación académica
- Historial laboral
- Idiomas
- Perfil profesional

## Tecnologías utilizadas

- HTML5
- Bootstrap 5.3.2
- CSS personalizado

## Visualización

Abrir el archivo `index.html` en un navegador web.
